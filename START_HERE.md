# Start here

This repository contains the implemented OnlyPaws BPSR guild tracker.

## For Codex

1. Confirm the repository is `Daa13x/BPSR-Guild-Tracker`.
2. Confirm the task-specific branch named by the current request.
3. Read `CODEX_TASK.md`.
4. Read every file under `docs/`.
5. Preserve the existing leaderboard, progression, achievement, event, reset, cache and audit logic.
6. Keep the supplied logo at `assets/guild-logo.png` unless an approved replacement is provided.
7. Implement, test, document, commit, push, and open or update only the requested draft PR.

## Scope boundary

Do not inspect or edit LifePlanSystem, LifePlanSystemPublic, or any other repository. Do not merge the PR.

## Repository safety

No secrets belong in Git. The optional emergency recovery secret, spreadsheet identifiers and runtime credentials remain in Google Apps Script Script Properties or ignored local configuration. Normal administration is authorized by `Players.IsAdmin` after member login.
