# Start here

This branch is the implementation workspace for the BPSR guild tracker.

## For Codex

1. Confirm the repository is `Daa13x/BPSR-Guild-Tracker`.
2. Confirm the branch is `agent/complete-guild-tracker`.
3. Read `CODEX_TASK.md`.
4. Read every file under `docs/`.
5. Extract the user-supplied `files.zip` and use its `Code.gs`, `Leaderboard.html`, and `README.md` as the baseline.
6. Add the supplied guild logo under `assets/` with a clear stable filename.
7. Inventory the baseline before editing and preserve working leaderboard logic.
8. Implement, test, document, commit, push, and open a draft PR to `main`.

## Scope boundary

Do not inspect or edit LifePlanSystem, LifePlanSystemPublic, or any other repository. Do not merge the PR.

## Repository safety

No secrets belong in Git. Admin secrets, spreadsheet identifiers, and runtime credentials must remain in Google Apps Script Script Properties or ignored local configuration.
